## Технологии

- React 18
- TypeScript
- Vite
- React Router
- Axios

## Установка

```bash
npm install
```

## Запуск

```bash
npm run dev
```

Приложение будет доступно по адресу `http://localhost:3000`

## Структура проекта

```
frontend/
├── src/
│   ├── api/           # API клиент и методы
│   ├── contexts/      # React контексты (Auth)
│   ├── pages/         # Страницы приложения
│   ├── types/         # TypeScript типы
│   ├── App.tsx        # Главный компонент
│   └── main.tsx       # Точка входа
├── index.html
├── package.json
└── vite.config.ts
```
